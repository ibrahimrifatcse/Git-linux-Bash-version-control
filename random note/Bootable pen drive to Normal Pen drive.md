#  How to convert Bootable Pendrive to Normal Pendrive?

###       open cmd

|command tutorial                            |
|--------------------------------------------|
|  type: >          diskpart       (in popop enter yes)         |
|  type: >          select disk n (n= your disk number)  |
|  type: >          clean                     |
|  type: >          create partition primary   |
|---------------------------------------------|
